# AI Transcripts

The assignment requires submitting the raw AI chat transcripts used during
development. This directory is where those transcripts will be collected for the
final submission.

## What goes here

- Exported chat logs from the AI coding tool(s) used (for example, the IDE agent
  conversation, including the design-decision discussions that shaped this
  library).
- Any supporting notes that show which suggestions came from the AI versus which
  decisions were made, corrected, or rejected by the project owner.

## How transcripts will be added

Transcripts will be exported near the end of the project and placed here as plain
text or Markdown files (one file per session, named by date/topic). If the tool
stores conversations locally, the locally available export will be included.

## Notes

- No transcripts are copied or fabricated at this stage; this file only documents
  the intended location and process.
- The structured record of design decisions (owner vs. AI) is kept in
  [`../docs/decisions.md`](../docs/decisions.md) and will be cross-referenced when
  assembling the transcripts.
